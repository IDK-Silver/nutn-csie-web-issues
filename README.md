# NUTN CSIE Website — Issue Tracker

Report problems with the **National University of Tainan, Department of Computer Science and Information Engineering (NUTN CSIE)** website here.

- Found a bug or something broken? **[Open a new issue](../../issues/new/choose)**.
- Please use the **Bug Report** template and fill in as much detail as you can (page, steps, screenshots). It helps us reproduce and fix it faster.

This repository is **for issue reporting only**. The website source code is maintained in a separate private repository.
